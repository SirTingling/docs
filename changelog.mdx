# Changelog

## v0.6.0 — Cloud Coordinator

Gateway-to-dashboard connectivity. Fleet management. Real-time policy streaming.

- **Gateway Coordinator** — Rust gRPC service connecting gateways to the dashboard
- **Fleet management** — Monitor connected gateway instances, track health, manage join keys
- **Policy streaming** — Push policy updates from dashboard to gateways in real-time
- **Audit shipping** — Gateway audit events flow to the dashboard database automatically
- **Health reporting** — Gateways send health reports every 30 seconds (version, sessions, SLI metrics, memory)
- **Stale detection** — Gateways that miss health reports are marked stale after 90 seconds
- **Deployment mode** — Organizations choose SaaS or On-Prem at creation

## v0.5.0 — Redis + Token Revocation

Distributed state. Instant key revocation. Circuit breaker sync.

- **Redis integration** — Distributed rate limiting across gateway instances
- **JWKS auto-discovery** — Gateways fetch signing keys automatically
- **Token revocation** — Revoked API keys propagate to all gateways within seconds
- **Circuit breaker sync** — Shared circuit state across gateway fleet via Redis

## v0.4.0 — Enterprise Infrastructure

Helm charts. CI/CD pipelines. Production-grade deployment.

- **Helm charts** — aegis-shield, gateway-coordinator, monitoring
- **CI/CD** — GitHub Actions for build, test, lint, security scan, Docker multi-arch builds
- **Container images** — Multi-arch (amd64/arm64), cosign-signed, Trivy-scanned
- **Monitoring** — Prometheus ServiceMonitor, Grafana dashboards, alert rules
- **Network policies** — Pod-level network isolation
- **Autoscaling** — HPA with configurable thresholds

## v0.3.0 — Gateway MVP

The Aegis Shield gateway. 11 modules. 369 tests.

- **Proxy engine** — Transparent HTTP/HTTPS proxy for AI provider APIs
- **Secret detection** — Real-time scanning of request bodies
- **Tokenization** — In-memory vault with deterministic token generation
- **Rehydration** — Automatic secret restoration in responses
- **Streaming support** — Full SSE streaming with chunk-level rehydration
- **Rate limiting** — Per-key, per-org rate limiting with sliding window
- **Circuit breakers** — Automatic fallback when upstream providers fail
- **Access logging** — Structured JSON logs (never logs secrets or bodies)
- **Auth module** — JWT validation, API key authentication
- **Health checks** — Liveness and readiness probes

## v0.2.0 — Core Engine

Detection, tokenization, and the in-memory vault.

- **Pattern detection** — 13+ secret types (AWS, Stripe, GitHub, database URLs, JWTs, etc.)
- **Context-based detection** — Generic secrets identified by surrounding code context
- **Config file parsing** — .env, YAML, JSON, TOML
- **Deterministic tokenization** — Same secret always produces the same token
- **Session vault** — Isolated in-memory storage, zeroized on completion
- **SHA-256 token generation** — Truncated hash-based tokens

## v0.1.0-alpha — Initial Release

First alpha. Aegis Shield CLI and API.

- **CLI** — `scan`, `tokenize`, `shield` commands
- **API** — `createSession()`, `session.tokenize()`, `session.rehydrate()`
- **Claude integration** — Built-in Anthropic API support
- **TypeScript** — Full type definitions
