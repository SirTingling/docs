# Installation

Takumo has two main components: the **CLI/API** for programmatic use and the **extension** for editor integration. Install whichever you need — or both.

---

## VS Code / Cursor Extension

The fastest way to get Takumo running. Install from your editor's marketplace.

<CodeGroup>

```bash VS Code
code --install-extension takumo.takumo
```

```bash Cursor
# Search "Takumo" in Cursor's extension panel
# Or install the universal VSIX from GitHub Releases
```

</CodeGroup>

Or search **"Takumo"** in the Extensions panel (`Cmd+Shift+X`).

The daemon binary downloads automatically on first activation. No manual setup needed.

## CLI / API (npm)

For programmatic use, CI/CD pipelines, and custom integrations.

<CodeGroup>
```bash npm
npm install github:SirTingling/takumo
```

```bash pnpm
pnpm add github:SirTingling/takumo
```

```bash yarn
yarn add SirTingling/takumo
```
</CodeGroup>

This gives you both the CLI and the API.

### From source

If you want to modify or contribute:

```bash
git clone https://github.com/SirTingling/takumo.git
cd takumo
pnpm install
pnpm build
```

Run commands from the repo:

```bash
pnpm aegis scan ./path/to/file.ts
pnpm aegis tokenize ./path/to/file.ts
pnpm aegis shield ./path/to/file.ts --prompt "your prompt"
```

### Global CLI install

To use `takumo-aegis` from anywhere:

```bash
cd takumo/packages/aegis
pnpm link --global
```

Now works globally:

```bash
takumo-aegis scan ~/any/project/src/
```

## Gateway (on-premises)

For teams deploying Takumo in their own infrastructure.

```bash
helm repo add takumo https://charts.takumo.io
helm install aegis-shield takumo/aegis-shield -n takumo
```

## Environment variables

| Variable | When needed | Description |
|----------|-------------|-------------|
| `ANTHROPIC_API_KEY` | `shield` command only | Your Claude API key |
| `TAKUMO_API_KEY` | Cloud features | Your Takumo API key |

The `scan` and `tokenize` commands work completely offline. Only `shield` needs an API key because it actually calls Claude.

## Verify it works

```bash
npx takumo-aegis --version
# takumo-aegis 0.1.0-alpha

npx takumo-aegis --help
# Usage: takumo-aegis <command> [options]
#
# Commands:
#   scan <path>       Find secrets in files
#   tokenize <file>   Show tokenized output
#   shield <file>     Full AI round-trip
```

## Requirements

- **CLI/API:** Node.js 20 or higher. macOS, Linux, or Windows (WSL recommended).
- **Extension:** VS Code 1.85+, Cursor, or any VS Code-compatible editor. macOS (Intel/Apple Silicon), Linux (x64/arm64), or Windows (x64).

<Accordion title="Node version issues?">
If you're on an older Node version:

```bash
# Using nvm
nvm install 20
nvm use 20

# Verify
node --version  # Should be v20.x.x or higher
```
</Accordion>
