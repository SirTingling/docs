## What You See on the Overview Page

The dashboard overview gives you an at-a-glance picture of your organization's security posture.

**Key metrics** are displayed at the top:

- **Total scans** — cumulative number of code scans processed
- **Secrets detected** — total secrets caught before they leaked
- **Sentinel issues** — security vulnerabilities and policy violations found
- **Active API keys** — how many keys are currently in use

**Scan statistics breakdown** shows the distribution of scan types across your organization — Shield scans, Sentinel reviews, code reviews, and policy evaluations.

**Policy compliance status** tells you how your organization is tracking against configured governance policies. Green means compliant. Yellow means warnings. Red means violations that need attention.

**Recent activity feed** surfaces the latest detections, key creations, member changes, and policy updates in chronological order.

**Threat trend visualization** charts detection volume over time so you can spot patterns — a spike in secret detections after onboarding a new team, a drop after tightening policies.

## Organizations

Each organization has its own dashboard, API keys, team, and billing. If you belong to multiple organizations, switch between them from the sidebar.

Your current organization context is always visible in the sidebar. All data — keys, scans, policies, audit logs — is scoped to the active organization.

## Roles

| Role | What it can do |
|------|---------------|
| **Owner** | Full access. Billing, members, settings, API keys, policies, streaming. Can delete the organization and transfer ownership. |
| **Admin** | Everything except deleting the organization, transferring ownership, and managing billing. Handles day-to-day operations. |
| **Security Lead** | Security settings, policies, audit log, fleet, compliance, and streaming (`streaming.read`, `streaming.write`). Cannot manage members or billing. |
| **Billing Admin** | Billing and plan management. Read-only access to the organization, members list, audit log, and notifications. No access to streaming, policies, or fleet. |
| **Member** | View dashboard, create API keys, triage issues, read streaming data. Cannot manage policies, members, or settings. |
| **Viewer** | Read-only access across the dashboard. Cannot create, edit, or delete anything. |

<Tip>Bookmark cloud.takumo.io/dashboard for quick access. The dashboard remembers your last active organization.</Tip>
